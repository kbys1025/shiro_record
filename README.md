# 城レコ

城レコはポートフォリオとして作られた、お城用のアルバムアプリです。

## 実装機能
* ユーザー登録/編集機能
* ユーザー一覧/詳細表示機能
* ユーザーフォロー機能
* ログイン/ログアウト機能
* 管理ユーザー機能
* ユーザー削除機能（管理ユーザー限定）
* コンテンツ投稿/編集/削除機能
* コンテンツ一覧/詳細表示機能
* 画像アップロード機能
* ページネーション機能
* 単体/統合テスト機能(minitest)

## 使用技術一覧
* 言語：Ruby, HTML, CSS, JavaScript
* フレームワーク：Ruby on Rails, Bootstrap
* ライブラリ：jQuery
* データベース：PostgreSQL
* バージョン管理：Git
* ソースコード管理：GitHub
* サーバー環境：Heroku
* 仮想環境構築：Vagrant
* 画像アップロード：CarrierWave
* 画像保存：Simple Storage Service(S3)
